# Profiles

Profiles let you save a complete set of appearance settings — colors, fonts, background image, and transitions — and switch between them instantly. Useful if you DJ at different venues or prefer different looks for different events.

![Profiles](https://raw.githubusercontent.com/richardsladetdj-creator/TangoDisplay/main/docs/screenshots/profiles.png)

---

## Built-in profiles

TangoDisplay ships with three read-only profiles:

| Profile | Description |
|---|---|
| **Classic** | Warm yellow/cream palette, traditional feel |
| **Modern** | Clean white and grey, minimalist |
| **High Contrast** | Black background with bright yellow text — maximum legibility at distance |

Click **Apply** to activate a built-in profile immediately.

---

## Custom profiles

Your saved profiles appear in the **Custom** section. Each profile row shows three color swatches (background, artist, genre) at the leading edge, followed by the profile name.

- The currently active profile is highlighted with an accent-color background and border, and marked with an **Active** badge
- Click **Apply** to switch to that profile
- Click the bin icon to delete a custom profile (built-in profiles cannot be deleted)

---

## Saving a new profile

1. Go to the **Appearance** tab and configure your colors, fonts, and background image
2. Scroll to the bottom and click **Save as New Profile…** (or **Save Current Settings As New Profile…** from the Profiles tab)
3. Enter a name and confirm

The new profile appears in the Custom section and is immediately set as active.

---

## Updating an existing profile

To overwrite a custom profile with current settings:

1. Make the profile **Active** (click Apply)
2. Adjust settings in the Appearance tab
3. Click **Save** (not "Save as New Profile")

This updates the active profile in place.
